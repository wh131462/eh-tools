# EH Tools

一款基于 uni-app 开发的多功能工具集小程序，支持微信、支付宝、H5 等多平台运行。

## 功能特性

### 时间工具
- **万年历** - 农历/公历日期查询
- **倒计时** - 自定义倒计时计时器
- **时间差计算** - 计算两个日期之间的时间差
- **世界时钟** - 全球主要城市时区显示
- **时间戳转换** - Unix 时间戳与日期互转
- **年龄计算器** - 精确计算年龄

### 计算工具
- **计算器** - 科学计算器
- **工资计算** - 税前税后工资换算
- **BMI 计算** - 身体质量指数计算
- **房贷计算** - 等额本息/等额本金计算
- **单位换算** - 长度、面积、体积等单位转换
- **个税计算** - 个人所得税计算

### 文本工具
- **编码转换** - Base64、URL 编码解码
- **二维码** - 二维码生成与扫描识别
- **JSON 格式化** - JSON 数据美化与校验

### 图片工具
- **图片压缩** - 图片体积压缩
- **颜色工具** - 颜色选择与格式转换

### 生活工具
- **大转盘** - 自定义选项的幸运转盘

## 技术栈

- **框架**: Vue 3 + TypeScript
- **跨平台**: uni-app
- **状态管理**: Pinia + 持久化插件
- **国际化**: Vue I18n (中/英文)
- **样式**: Sass + 主题切换 (亮色/暗色)
- **其他**: crypto-js, qrcode-generator, lucky-canvas

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发调试

```bash
# H5
pnpm dev:h5

# 微信小程序
pnpm dev:mp-weixin

# 支付宝小程序
pnpm dev:mp-alipay
```

### 构建发布

```bash
# H5
pnpm build:h5

# 微信小程序
pnpm build:mp-weixin

# 支付宝小程序
pnpm build:mp-alipay
```

## 项目结构

```
src/
├── components/     # 公共组件
├── i18n/          # 国际化语言包
├── pages/         # 页面
│   ├── index/     # 首页
│   ├── calc/      # 计算工具
│   ├── time/      # 时间工具
│   ├── text/      # 文本工具
│   ├── image/     # 图片工具
│   └── life/      # 生活工具
├── static/        # 静态资源
├── store/         # Pinia 状态管理
├── styles/        # 全局样式
├── types/         # TypeScript 类型定义
└── utils/         # 工具函数
```
